# pony-tictactoe

a Tic-Tac-Toe implementation in pony.

## building

```
$ cd /path/to/project
$ ponyc
```

## running

```
$ ./tic-tac-toe
TicTacToe starting...
   1 2 3
A [ , , ]
B [ , , ]
C [ , , ]
```

Players take turns and enter the coordinate where they want to place their marker, e.g.

`A1` places a circle on top left corner.
